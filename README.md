
# Exam Management System

An exam management system build for schools and colleges in mind. Authority can hold exams with unique exam ID and get auto generated admit cards as well as results publication features.

## Tech Stack

**Client:** Nextjs, TailwindCSS

**Server:** Nextjs

**Database:** Mongodb


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`COLLEGE_USER_NAME`

`COLLEGE_PASSWORD`

`DB(database name)`

`Mongodb URI`



## Features

- Exam creation with automatic unique exam ID generation
- CRUD operations for students data
- Auto Admit Cards generation
- Simple result saving
